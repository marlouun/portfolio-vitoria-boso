COMO RODAR NO VS CODE

1. Abra esta pasta no VS Code.
2. Abra o terminal.
3. Rode:

npm install
npm run dev

4. Abra:

http://localhost:5173

COMO TESTAR A VERSAO FINAL

npm run build
npm run preview

OTIMIZACAO

Esta versao esta mais leve porque as animacoes nao dependem mais do framer-motion.
As animacoes principais usam CSS e IntersectionObserver.
As imagens abaixo da primeira dobra usam loading="lazy".
