CORRECAO DO ERRO NO GITHUB ACTIONS

O erro "Cannot find module 'react'" / "Cannot find module 'vite'" acontece quando o build roda sem as dependencias instaladas no ambiente do GitHub.

Arquivos importantes que precisam estar no repositorio:
- package.json
- package-lock.json
- vite.config.ts
- .github/workflows/deploy.yml
- src/
- public/

Depois de substituir os arquivos do projeto por esta versao, rode:

git status
git add .
git commit -m "Corrige dependencias e deploy do GitHub Pages"
git push

No GitHub, confira:
Settings > Pages > Source: GitHub Actions

Depois va em Actions e acompanhe o workflow.
